# Environment variables template

Copy to `.env` in project root (or `api/.env`). **Never commit real values.**

```bash
# ═══════════════════════════════════════════════════════════
# KICKOFF — Hackathon core (required for local backend)
# ═══════════════════════════════════════════════════════════
PORT=3001
PEARS_PORT=3002
TEAM_NATION=Ghana

# QVAC — local inference, NO API KEY
# Verify model ID against: https://docs.qvac.tether.io
QVAC_MODEL=llama-3.2-3b-instruct

# WDK — testnet for hackathon demo
WDK_NETWORK=testnet

# ═══════════════════════════════════════════════════════════
# Live match data (optional — online only, NOT for AI inference)
# Get key: https://agent.tinyfish.ai/api-keys
# ═══════════════════════════════════════════════════════════
TINYFISH_API_KEY=

# ═══════════════════════════════════════════════════════════
# DEV ONLY — do NOT wire to /api/ai/analyze (QVAC track violation)
# ═══════════════════════════════════════════════════════════
VENICE_API_KEY=
OPENAI_API_KEY=

# ═══════════════════════════════════════════════════════════
# Azure (optional — hosting / Key Vault, NOT required for hackathon)
# ═══════════════════════════════════════════════════════════
AZURE_SUBSCRIPTION_ID=
AZURE_TENANT_ID=
AZURE_CLIENT_ID=
AZURE_CLIENT_SECRET=
AZURE_KEY_VAULT_URL=
```

See [03-API-KEYS-MATRIX.md](./03-API-KEYS-MATRIX.md) for full rationale.
